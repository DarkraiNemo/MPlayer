# MPLayer - Music Player with Electron

* Welcome to MPlayer, a native Player Music to jam along!

---

## The purpose of the project:

1. MPlayer brings to the User listen *(for now)*, any type of formatted song!
2. Drop your folder, see the visualizer work on.
3. Very customizable *(NOT IMPLEMENTED YET.)*!

## Features to be implemented:

1. New Player Menu, being more dynamic;
2. New type of Drop, being on the entire website instead of a dedicated button;
3. New types of Visualizer, being customizable with colors and stuff;
4. Work for small resolutions, still buggy at the moment;

## Please note:

* This is my first project, so things *will* break out! Being like: menu not functioning properly, some menus not fitting the content, and so on. I will be making some updates for him to work properly, so don't worry about that!

## How do I download this project?

* Simple, go to the **Releases** tab! It *should* have two files on it:
    1. The Windows version:
        1.1. Just click it! Ain't that easy?
    2. The Linux version, also with a installer and things;
        2.1. On the installer, make sure to know where the file is;
        2.2. Then, execute the program as:
        ```
        chmod +x MPlayer-1.0.0.AppImage
        ./MPlayer-1.0.0.AppImage
        ```
        *(./ is to run the program.)*
